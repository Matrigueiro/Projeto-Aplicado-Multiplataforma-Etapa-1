## Descrição do Projeto
Este projeto consiste em uma plataforma digital para registro, acompanhamento e resolução de demandas comunitárias, permitindo que cidadãos reportem problemas urbanos e acompanhem sua resolução.



## Problema e Justificativa
Muitas comunidades enfrentam dificuldades para comunicar problemas como infraestrutura, iluminação e limpeza urbana. A solução proposta centraliza essas demandas, facilitando a comunicação entre cidadãos e responsáveis.



## Objetivos
- Facilitar o registro de demandas comunitárias
- Permitir acompanhamento do status
- Priorizar problemas com base em apoio popular
- Melhorar a comunicação entre população e responsáveis



## Escopo
O sistema inclui:
- Cadastro e autenticação de usuários
- Registro e gerenciamento de demandas
- Sistema de apoio às demandas
- Comentários e feedback
- Painel administrativo



## Arquitetura
O sistema segue arquitetura cliente-servidor em camadas:

- Frontend: React / React Native
- Backend: Node.js + Express
- Banco de Dados: PostgreSQL



## Tecnologias
- React.js / React Native
- Node.js
- Express
- PostgreSQL
- JWT (autenticação)
